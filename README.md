# BackgroundWorkAround

En: Example of the most survival foreground service on Android without special rights. For demonstration of work plays sound and restarts at reboot (if system does not block autostart of boot receivers).

Ru:
Пример самого живучего foreground сервиса на Android без специальных прав. Для демонстрации работы вопроизводит звук и перезапускается при перезагрузке (если система не блокирует автозапуск boot ресиверов).
